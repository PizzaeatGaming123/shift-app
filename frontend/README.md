# 暁夢シフト管理アプリ（デモ）

ラーメン店「株式会社暁夢」（岡山県倉敷市・3店舗）向けのシフト管理デモ。
ブラウザだけで動作します（サーバー・ログイン不要）。

## 起動

```bash
npm install
npm run dev
```

表示された `http://localhost:5173` をブラウザで開きます。

## テスト

```bash
npm test
```

## 機能（デモ範囲）

1. **スタッフの希望提出** — カレンダーをタップして 早番(7:00-16:00) / 遅番(15:00-24:00) / 早番+遅番 / 休み希望 を選ぶ
2. **店長の確認・割り当て** — 全スタッフの希望を一覧表で確認し、希望セルをタップで割り当て。各日の人数と充足（不足=赤/適正=緑/過多=橙）が見える
3. **確定シフトの共有** — 確定したシフトを、誰がいつ出勤かわかるカレンダーで表示

## 使い方

- 画面上部の **店舗選択**（中島店／新田店／早島店）と **役割切替**（スタッフ用／店長用）で表示を切り替えます。
- データはブラウザの localStorage に保存されるため、リロードしても保持されます。

## 技術構成

React + Vite + TypeScript。状態管理は React Context + useReducer、永続化は localStorage。
ロジック（日付・希望・割り当て・充足判定）は Vitest でテスト済み。
